# Mero Board

Mero Board is a versatile Kanban-style project and task management system. It enables teams to visualize their work pipelines, collaborate in real-time, and manage complex project hierarchies through boards, lists, and tasks.

## 🚀 Standard Features
- **Kanban Boards**: Drag-and-drop workflow visualization.
- **Workspaces**: Group related boards by department or project.
- **Real-time Collaboration**: Instant updates and @mentions.
- **Task Hierarchy**: Checklists, attachments, and sub-tasks.
