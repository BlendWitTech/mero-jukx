# Mero Inventory

Mero Inventory is a robust inventory management system designed for multi-location businesses. it handles product lifecycles, real-time stock tracking, purchase/sales workflows, and barcode-based operations.

## 🚀 Standard Features
- **Multi-Location Support**: Track stock across warehouses and branches.
- **Product Variants**: Manage SKUs with different sizes, colors, and attributes.
- **Order Management**: Integrated Purchase Orders (PO) and Sales Orders (SO).
- **Barcode/QR Support**: Scan for quick stock adjustments and lookups.
