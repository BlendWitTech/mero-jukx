# Mero Khata

Mero Khata is a simplified accounting system designed for small businesses and secondary organizations in Nepal. It focuses on ease of use, providing essential features like income/expense tracking, basic invoicing, and VAT management without the complexity of full double-entry bookkeeping.

## 🚀 Standard Features
- **Simplified Accounting**: Pre-configured Chart of Accounts for Nepal SMEs.
- **Quick Invoicing**: Professional bills with automated 13% VAT calculation.
- **Income/Expense Tracking**: Mobile-friendly transaction entry.
- **VAT Summary**: Reports prepared for IRD (Inland Revenue Department) submission.
