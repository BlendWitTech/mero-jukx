# Mero HR

Mero HR is a comprehensive human resource management system designed for the Nepali market. It handles employee records, attendance tracking, leave management, and payroll compliance specifically for Nepal.

> [!NOTE]
> This module is currently in the **Planned/Development** phase. The documentation below reflects the intended architecture and features.

## 🚀 Planned Features
- **Employee Management**: Centralized digital employee records and document repository.
- **Attendance & Leave**: Support for biometric integration and Nepal public holidays.
- **Payroll (Nepal SSF)**: Automated salary calculation with SSF, CIT, and Tax slabs.
- **Self-Service Portal**: Employee portal for payslips and leave requests.
