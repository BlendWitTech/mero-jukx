# Mero CRM

Mero CRM is a comprehensive customer relationship management system designed specifically for the Nepali business ecosystem. It handles the entire sales lifecycle from lead generation to deal closure and post-sales activity tracking.

## 🚀 Standard Features
- **Lead Management**: Capture and score leads from various sources.
- **Sales Pipeline**: Visual Kanban board to track deal progress.
- **Contact Management**: 360-degree view of customers, vendors, and partners.
- **Activity Tracking**: Log calls, emails, meetings, and tasks.
