# Mero CMS

Mero CMS is an integrated content management system that allows organizations to manage their web presence, blogs, and basic e-commerce directly from the ERP platform.

## 🚀 Key Features
- **Website Builder**: Drag-and-drop interface for creating responsive pages.
- **Integrated Blog**: Full rich-text editor with SEO metadata support.
- **Lead Capture**: Forms that feed directly into the Mero CRM module.
- **Media Management**: Centralized library for images and documents.
