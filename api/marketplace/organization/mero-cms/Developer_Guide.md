# Mero CMS - Developer Guide

## Theme Development
Themes are stored as JSON templates. developers can add new theme definitions in the `api/marketplace/organization/mero-cms/themes` directory.

## Lead Capture Integration
Ensure that all forms generated by the CMS include the `crmSync` flag to automatically create leads in the **Mero CRM** module upon submission.

## Best Practices
- Optimize images automatically on upload.
- Use CDN paths for all media delivery.
- Ensure SEO tags are strictly validated to prevent malformed meta headers.
