# Mero Accounting

Mero Accounting is an enterprise-grade, double-entry accounting system compliant with Nepal Financial Reporting Standards (NFRS). It is designed for medium to large organizations requiring advanced financial controls, multi-currency support, and deep reporting.

## 🚀 Standard Features
- **Double-Entry General Ledger**: Full audit trails and journal automation.
- **NFRS Compliance**: Standardized Trial Balance, P&L, and Balance Sheet.
- **Advanced Accounts Receivable/Payable**: Aging reports and automated dunning.
- **Fixed Assets**: Automated depreciation and asset registry.
